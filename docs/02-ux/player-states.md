# Player States

## Surfaces

- Lobby preparation.
- Annonce captivante.
- Pret/non pret.
- Mini-jeu actif.
- Fin de manche et attente.
- Reconnexion.
- Resultats publies.
- Elimination/spectateur.

Chaque surface doit avoir un message fonctionnel precis. Les textes vagues comme `En attente du serveur...` sont interdits comme etat final.
