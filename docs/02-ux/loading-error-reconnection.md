# Loading Error Reconnection

## Loading

Un loading doit dire ce qui est charge: participation, lobby, room, snapshot, mini-jeu, resultats publies.

## Erreurs

Chaque erreur expose un code produit: non participant, non autorise, partie non active, live indisponible, reconnect expired.

## Reconnexion

Le joueur doit savoir:

- si sa place est conservee;
- si son input a ete recu;
- s'il est en attente de verification;
- quand contacter support.
