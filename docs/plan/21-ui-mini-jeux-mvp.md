# Feature 21 - UI des mini-jeux MVP

## Objectif sprint

Rendre jouables les mini-jeux MVP avec les runtimes server-side de Feature 18.

## Dependances

- Feature 18.
- Feature 20.

## Gate documentaire obligatoire

1. Context7 PixiJS v8 uniquement si un jeu plateau/canvas est retenu.
2. Les jeux grille/boutons restent en React + RetroUI + Motion.

## User stories

Pour chaque jeu MVP : consommer les messages runtime, envoyer les actions avec nonce, desactiver input apres soumission, afficher accepted/rejected sans scoring client.

## Definition of Done

Trois jeux minimum jouables en session complete multi-joueurs, test automatise de non-fuite de reponse sensible.
