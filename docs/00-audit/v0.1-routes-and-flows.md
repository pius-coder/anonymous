## ConnectRPC (`apps/api/src/rpc/`)

Services enregistres :

```text

RealtimeV1.RealtimeAccessService  → realtime-service.ts
```

Scoring, MiniGame, Admin, Notification.

## Game server (WebSocket Colyseus)

| Element | Valeur |
|---|---|
| Room type | `game_room` |
| Filtre matchmaking | `partyId` |
| Port defaut | 3002 (`apps/game-server` config) |
| Auth | Token live hash en DB |

### Messages client → serveur

| Commande | Handler | Role |
|---|---|---|
| `room:move` | movement-handler | Joueur |
| `round:submit` | round-handler | Joueur |
| `round:finish` | round-handler | Joueur |
| `snapshot:request` | readonly-handler | Admin / observer |

### Cycle de vie connexion

```text
Client → POST /v1/live/parties/:partyId/access (API)
       → token + ws URL
       → Colyseus joinOrCreate("game_room", { partyId })
       → GameRoom.onAuth (validateLiveToken)
       → GameRoom.onJoin (addPlayer, persist connected)
       → simulation tick + sync schema
       → onLeave → fenetre reconnexion 30s
       → onDispose → cleanup
```

## Flux metier cibles (etat branchement)

### Flux A — Inscription joueur (API pret, UI mock)

```text
Catalogue → Detail partie → Participation → Paiement → Lobby
     ↓            ↓              ↓             ↓          ↓
  GET /parties  GET /:code   POST register  POST pay   preparation/*
  [mock UI]     [mock UI]    [REST OK]      [REST OK]  [REST OK / mock UI]
```

### Flux B — Live social room (partiel)

```text
Lobby pret → Room Phaser → Mouvement → Round → Waiting → Results
     ↓           ↓            ↓          ↓        ↓         ↓
 confirm-start  live access  room:move  round:*  UI etat   mock
 [admin REST]   [RPC+WS OK]  [GS OK]    [partiel] [UI]     [mock]
```

### Flux C — Admin command center (UI > API)

```text
Setup → Publish → Open preparation → Start round → Monitor → Scores → Publish
  ↓        ↓            ↓                ↓           ↓         ↓         ↓
 admin   admin        admin            admin       mock      mock     absent
 REST    REST         REST             REST
```

### Flux D — Finance / support (stubs UI)

