## Comparaison HEAD legacy → v0.1

| Aspect | HEAD | v0.1 |
|---|---|---|
| Room Colyseus | `GameSessionRoom` monolithique 993 lignes | `GameRoom` focalisee social + round |
| Vocabulaire | "session" omnipresent | "party" domaine |
| Contrats | Zod + Prisma + hooks disperses | Protobuf central + REST transition |
| Mini-jeux | 3 runtimes + 36 definitions | 0 runtime, catalogue doc seulement |
| UI live | `/live` universel | Parcours etats explicites (lobby, room, round, waiting) |
| Melange admin/joueur | Eleve | Separe par layouts et routes |

Reprise legacy : voir `feature-implementation-matrix.md` et `head-forensic-audit.md`.
