# Safe Removal

1. Identifier fichiers.
2. Documenter fonction actuelle.
3. Documenter dependances.
4. Documenter remplacement.
5. Evaluer risque.
6. Obtenir validation explicite.
7. Supprimer en branche dediee.
8. Lancer typecheck/lint/test/build.
9. Documenter resultats.

Les licences, lockfiles et configurations indispensables ne sont jamais supprimes silencieusement.
