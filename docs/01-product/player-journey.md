# Player Journey

## Etats joueur

- Invite.
- Inscrit/paye.
- Preparation ouverte.
- Present.
- Pret.
- En manche.
- Manche terminee, attente verification.
- Resultats publies.
- Qualifie, elimine, abandon, deconnecte.

## Fin de manche

Apres avoir termine:

- l'UI indique que la manche est terminee;
- les resultats sont en verification;
- le joueur peut lire le message admin;
- le lobby de discussion est disponible si autorise;
- aucun score non publie n'est visible.
