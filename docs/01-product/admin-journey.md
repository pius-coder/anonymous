# Admin Journey

## Parcours cible

1. Ouvrir le centre de controle de la partie.
2. Voir participants invites, hors ligne, connectes, presents, prets, sans reponse.
3. Envoyer annonce pre-match et notification associee.
4. Voir livraison, affichage ou confirmation quand disponible.
5. Confirmer explicitement le lancement.
6. Superviser manche active globalement.
7. Ouvrir une vue individuelle lecture seule.
8. Verifier scores provisoires.
9. Corriger selon regles documentees.
10. Publier les resultats.
11. Choisir manche suivante ou fin.

Le centre admin doit separer `observer`, `decider`, `publier`.
